Type Changes By
Added action to close versioning diff editor tabs. Fixes 22. bahlef

Release 1.0.4 – 2023-08-10

Type Changes By
Fixed new VCS and project actions. Fixes 17. bahlef

Release 1.0.3 – 2023-06-15

Type Changes By
Added close action to close all tabs which are not changed in the used VCS (Git, SVN and Mercurial supported). Fixes 13. bahlef
Added close actions to close all tabs which either belong or not belong to the project of the selected tab. bahlef

Release 1.0.2 – 2023-01-26

Type Changes By
Signing of nbm file as requested by the NetBeans team to get the plugin center verification bahlef

Release 1.0.1 – 2022-10-02

Type Changes By
With the help of Yenta, I was able to move the actions right next to the NetBeans close actions. Fixes 1. bahlef

Release 1.0.0 – N/A

Type Changes By
Update plugin to latest dependencies and plugins and made it ready for the NetBeans plugin center to be published bahlef
